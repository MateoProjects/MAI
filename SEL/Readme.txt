###### Folders description #####
################################

* Source folder. Contains the source code of the project.
* Data folder. Contains the data of the project. Inside can be found all datasets neded
* Results folder. Contains the results of the project.

################################
###### How to execute code #####
################################

1. Go to Foler Rules Classifier
2. Execute the following command in terminal:  python3 .\Source\main.py    
3. The execution show the following:
    Insert 0 for mushrooms dataset, 1 for fire dataset, 2 for balance dataset else -1: 

4. Insert the number that the dataset you want to use. 
5. The results will appear in the terminal in a few seconds.
